## 🛠 開発時の注意点

### 1. CSVフォーマットについて
- ヘッダー行はありません（1行目からデータ）。
- 列順序: `店舗CD`, `商品CD`, `発注数`

### 2. モードによる挙動の違い
- **Morningモード（朝締め）** の場合、発注日（`oddt`）は強制的に `NULL` になります。
- テスト時はDBサーバーの時間に注意してください（`Get_DB_Time` を使用しています）。
